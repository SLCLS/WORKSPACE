# DEBUGGING INFORMATION
This file explains some of the values used for this program, useful for debugging or adapting the code into a new device/resolution.

## CIRCLE VALUES:
**Dark mode theme is used for this Detection system.** The 2 Digits number represent the RGB Color Integer of the circles, this are interpreted by the Puzzle Solver as `0` or `1`.

### RGB Color Integer
    0  = Dark Colored (UP ARROW)
    68 = Light Colored (DOWN ARROW)

### Values Interpretation
    Dark = `0`
    Light = `1`
